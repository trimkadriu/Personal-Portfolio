JE.NET Cooperation

JE.NET is a database of Junior Enterprises (JEs) affiliated to Brasil Júnior and JADE in order to promote and facilitate the interaction amongst JEs from Europe and Brazil.

Some of the technologies that are used in this projects

Project description

JE.NET is a database of Junior Enterprises (JEs) affiliated to Brasil Júnior and JADE in order to promote and facilitate the interaction amongst JEs from Europe and Brazil. Using this tool JEs will be able to contact any other JE in the world that is willing to cooperate with another JE.

Implementation description

JE.NET is developed with CodeIgniter MVC framework written in PHP language, deployed in Apache HTTP Server, and use MySQL as a database.
